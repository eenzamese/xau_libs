## About The Project

Just another QUIK processing library for internal projects.

## Installation

Clone the repo
   ```sh
   git clone https://github.com/eenzamese/xau_libs
   ```
Don't forget to pull submodules
   ```sh
   git submodule update --init recursive
   ```
